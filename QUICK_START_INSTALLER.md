# Быстрый старт: Создание установщика Windows

## Простейший способ

1. **Запустите скрипт сборки:**
   ```bash
   # Windows CMD
   build_installer.bat
   
   # Или PowerShell
   .\build_installer.ps1
   ```

2. **Найдите установщик:**
   После успешной сборки установщик будет находиться в:
   ```
   build\windows\x64\runner\Release\iptvca_[версия]_x64.msix
   ```

3. **Установите сертификат (ОБЯЗАТЕЛЬНО!):**
   ```powershell
   # Запустите PowerShell от имени администратора
   .\install_certificate.ps1
   ```
   
   **ВАЖНО:** Без установки сертификата кнопка "Install" будет неактивна!

4. **Установите приложение:**
   Дважды кликните на файл `.msix` для установки. После установки сертификата кнопка "Install" будет активна.

## Ручная сборка

Если скрипт не работает, выполните команды вручную:

```bash
# 1. Установите зависимости
flutter pub get

# 2. Соберите приложение
flutter build windows --release

# 3. Создайте установщик
dart run msix:create
```

## Альтернатива: NSIS установщик (без сертификатов)

Если вы не хотите возиться с сертификатами, используйте NSIS:

1. **Установите NSIS:**
   - Скачайте с https://nsis.sourceforge.io/Download
   - Установите NSIS

2. **Создайте установщик:**
   ```bash
   # Windows CMD
   build_nsis_installer.bat
   
   # Windows PowerShell
   .\build_nsis_installer.ps1
   ```

3. **Найдите установщик:**
   Файл `IPTVCA_Setup_1.0.0.exe` будет в корне проекта

**Преимущества NSIS:**
- ✅ Не требует установки сертификатов
- ✅ Работает на всех версиях Windows
- ✅ Простая установка для пользователей

Подробнее см. `NSIS_README.md`

## Примечания

- **MSIX** - современный формат, но требует установки сертификата
- **NSIS** - традиционный .exe установщик, работает без сертификатов
- Для продакшена рекомендуется использовать подписанный сертификат (см. INSTALLER_README.md)
